
usage: httpc get [-v] [-h key:value] [-o name] URL

Get executes a HTTP GET request for a given URL.

          -v                 Prints de details of a response such as protocols, status and headers.

          -h key:value       Associates Headers to HTTP Request with the format
                             'key:value'.

          -o                 Write the body of the response to the specified file instead of the console.


Arguments can be in any order.
